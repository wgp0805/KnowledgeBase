## 核心摘要
小锋（java1234）撰写的 LangChain4j 全面教程，从设计理念到核心架构到工程实践。LangChain4j 是 Java 世界首款成熟的 LLM 应用框架，灵感来自 Python LangChain 但非简单翻译，充分尊重 Java 工程习惯（强类型、注解驱动、依赖注入）。核心抽象七大件：ChatLanguageModel、AI Services（声明式接口）、ChatMemory、Tools/Function Calling、RAG、Streaming、结构化输出。架构分五层：应用层→高级 API 层→核心抽象层→集成层→基础设施层，每层只依赖下层抽象。
