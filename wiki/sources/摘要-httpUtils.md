## 核心摘要
基于 JDK 原生 HttpURLConnection 和 HttpsURLConnection 封装的 HTTP 请求工具类，支持 GET/POST、自动重试机制和 HTTPS 证书信任处理。

## 关联连接
- HttpURLConnection — Java HTTP 客户端
- HttpsURLConnection — Java HTTPS 客户端
