## 核心摘要

Elasticsearch 7.11 保姆级入门教程。文章从 ES 概述与历史讲起，对比了 ES 与 Solr 的差异，详细演示了 ES 在 Linux 上的安装配置、elasticsearch-head 和 Kibana 的部署。深入讲解了 ES 的核心概念（文档、类型、索引、分片、倒排索引），IK 中文分词器的安装与自定义词库，最后通过大量 RESTful API 示例演示了索引和文档的增删改查、排序分页、布尔查询、精确查询和高亮显示等操作。
