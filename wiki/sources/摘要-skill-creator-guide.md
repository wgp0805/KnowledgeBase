## 核心摘要
skill-creator 是 Claude Code 官方提供的元技能（Meta-Skill），专门帮助用户通过自然语言对话来自动化创建和优化 AI 技能。该工具覆盖技能开发的完整生命周期：从需求分析、规则编写到频率控制的工作流自动化，再到边界逻辑优化、触发精准度优化和 A/B 基准测试验证。安装方式为从 GitHub `anthropics/skills` 仓库复制到本地 `~/.claude/skills/` 目录。应用场景涵盖 Git 提交规范、代码审查标准、文档生成、业务报告自动化、Excel/PDF/Word 结构化提取以及公司 SOP 生成等。
