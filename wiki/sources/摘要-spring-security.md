## 核心摘要
Spring Security 完整教程，涵盖底层架构（DelegatingFilterProxy、FilterChainProxy、SecurityFilterChain）、用户认证（内存/数据库）、密码加密（BCrypt/Argon2）、前后端分离配置（JSON 响应、跨域 CORS）、会话并发控制、授权管理（request/method 级别）、OAuth2 四种授权模式及 GitHub 社交登录案例。
