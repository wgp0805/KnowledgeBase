## 核心摘要

这篇文章介绍了4个用于Claude Code的Spring生态专属AI Skill开源项目。这些技能可以让Claude Code生成符合Spring Boot 4/Spring Framework 7最新规范的企业级代码。四个项目分别是：dr-jskill（JHipster作者出品的企业级项目脚手架）、agent-skill-java-spring-framework（强制使用Spring最新API规范）、sivalabs-agent-skills（统一AI技能封装规范）、spring-testing-skills（自动生成全套测试用例）。文章展示了6个实战用法，包括项目生成、规范约束、业务封装、测试生成和代码重构。
