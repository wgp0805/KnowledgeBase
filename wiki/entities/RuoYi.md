## 定义
RuoYi（若依）是一个基于 Spring Boot + MyBatis 的 Java 快速开发框架，内置 RBAC 权限管理、代码生成器等功能。

## 关键信息
- 提供完整的管理后台功能：用户管理、角色管理、菜单管理、部门管理
- 内置代码生成器，可快速生成 CRUD 代码
- 支持多数据源和缓存集成
- 前后端分离版本（RuoYi-Vue）使用 Vue 3 + Element Plus
