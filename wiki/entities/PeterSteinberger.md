## 定义

Peter Steinberger，绰号"龙虾之父"（来自其爆火的 AI 编程项目 OpenClaw，claw 意为龙虾钳），知名 iOS 开源开发者、PSPDFKit 创始人，后加入 OpenAI。因在 X 上晒出月烧 940 万元 token 的账单而引发广泛关注。

## 关键信息

- **身份**：OpenClaw 项目创始人、PSPDFKit 创始人、前 OpenAI 员工
- **极端案例**：
  - 过去 30 天调用 OpenAI API 总费用 130 万美元（约 940 万元人民币）
  - 消耗 6030 亿 token，发起 760 万次请求
  - 每日请求量 20.6 万次，每秒调用 2.4 次
- **成本构成**：开了"快速模式"，让上百个 Codex 高频并发跑推理；关掉快速模式成本能降 70%
- **团队规模**：只有 3 人，但同时在云端运行约 100 个 Codex
- **成本逻辑**：不是拿 Codex 和"工具成本"比，是拿它和"工程团队成本"比——一个月 token 账单约等于传统时代 25-30 人工程师团队的人力成本
