## 定义
JHipster 是一个开源的应用生成器，用于快速搭建基于 Spring Boot + 前端框架（Angular/React/Vue）的全栈 Web 应用，内置最佳实践和企业级特性。

## 关键信息
- 生成内容：Spring Boot 后端 + 前端 SPA + Docker 配置 + CI/CD 配置
- 技术栈：Spring Boot、Spring Security、JPA/Liquibase、前端框架
- JDL（JHipster Domain Language）：用声明式语法定义实体和关系
- 微服务支持：网关、注册中心、微服务应用生成
- dr-jskill：JHipster 作者新作，Claude Code 的企业级脚手架技能
