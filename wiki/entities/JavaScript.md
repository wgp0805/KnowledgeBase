## 定义
JavaScript 是一种轻量级的解释型编程语言，主要运行在浏览器端，也可通过 Node.js 在服务端运行，是 Web 开发的三大核心技术之一。

## 关键信息
- async/await：基于 Promise 的异步编程语法糖
- async 函数返回 Promise 对象
- await 暂停函数执行直到 Promise 完成
- 相比 Promise then 链，async/await 使异步代码更接近同步代码风格
- 异常处理使用 try/catch 包裹 await 调用
