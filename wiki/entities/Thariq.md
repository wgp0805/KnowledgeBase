## 定义
Anthropic 公司工程师，Claude Code Skills 内部使用者，分享了 Skills 的 9 类分类法和设计经验。

## 关键信息
- 在 Anthropic 内部使用数百个 Claude Code Skills
- 将 Skills 归纳为 9 大类别：知识、验证、数据访问、自动化、脚手架、代码审查、部署、调试、运维
- 提出 Skills 应是文件夹而非单个文件的设计理念
- 强调 Gotchas（踩坑经验）是 Skill 中含金量最高的部分
- 建议 description 字段应作为触发条件而非功能说明
- 提倡通过 Helper Functions 让 Claude 专注于决策而非样板代码
- 介绍了 Skills 的跨会话记忆存储机制
- 分享了团队 Skills 分发和管理的最佳实践
