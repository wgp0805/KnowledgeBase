## 定义
JHipster 作者 Julien Dubois 开发的企业级项目脚手架技能，专门赋能 Claude Code 生成生产可用、对标 JHipster 的 Spring Boot 完整项目。

## 关键信息

### 核心能力
- 自动生成 Spring Boot 4.x + Java 25 项目
- 集成 PostgreSQL、Docker 镜像、多前端架构（Vue/React/Angular）
- 包含监控告警、生产级配置
- 自动关闭冗余代理、开启 JSpecify 非空校验

### 使用方式
将 dr-jskill 导入 Claude Code 技能目录，直接输入指令：
> 基于dr-jskill规范，创建一个生产级SpringBoot4后台项目，集成PostgreSQL、Docker、监控链路、Vue前端

### 项目地址
https://github.com/jdubois/dr-jskill
