## 定义
FreeMarker 是一个基于模板和数据模型生成文本输出的 Java 模板引擎，常用于生成 HTML 页面、邮件模板、配置文件等，Spring Boot 内置支持。

## 关键信息
- 模板语言：${变量}、<#if>、<#list>、<#assign> 等指令
- 数据模型：Java 对象通过 TemplateModel 接口暴露给模板
- 与 Thymeleaf 对比：FreeMarker 功能更强大，Thymeleaf 更易上手
- Spring Boot 整合：spring-boot-starter-freemarker 自动配置
- 文件后缀：.ftl（FreeMarker Template Language）
