## 定义
Windows 是微软开发的操作系统系列，是全球市场份额最大的桌面操作系统，广泛用于个人电脑和企业环境。

## 关键信息
- 最新版本：Windows 11
- WSL2：Windows Subsystem for Linux，可在 Windows 上运行 Linux 环境
- PowerShell：命令行脚本环境
- Hyper-V：内置虚拟化技术
- Docker Desktop：通过 WSL2 后端运行容器
