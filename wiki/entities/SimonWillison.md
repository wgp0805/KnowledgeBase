## 定义

Simon Willison，Web 框架 Django 的联合创始人，在开发者圈极具影响力的独立开发者和博主。他提出了 **Vibe Engineering** 这一概念，专门用来与 Vibe Coding 划清界限。

## 关键信息

- **身份**：Django 联合创始人、独立开发者、技术博主
- **核心贡献**：提出 Vibe Engineering 概念
- **Vibe Engineering 定义**：对每一行代码负责，但在规划、架构、调试每个环节都重度调度 Agent
- **与 Vibe Coding 的区别**：
  - Vibe Coding：让模型随便写，然后祈祷测试能过（甩手掌柜）→ 被替代
  - Vibe Engineering：保持人类责任，重度调度 Agent → 被放大
